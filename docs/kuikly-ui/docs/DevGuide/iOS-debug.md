# iOS平台调试

ios平台有两种方式可供调试
1. 使用Android Studio在kmp工程调试: 适合在demo验证期使用
2. 安装第三方插件，在xcode调试: 适合在iOS宿主侧使用

## 使用Android Studio调试

这种方式是复用Android Studio本身的调试能力

![ios调试](./img/ios_as_debug.png)
